# API Wrapper for OGC API EDR
## How to use

```
edr = EDRWrapper("http://example.org/v1.0/")

# show functions of EDRWrapper
edr.help

# example
edr.setCollectionId("ab12cd")
edr.getCollection()
```

## API provided in EDR


1. https://ogcapi.ogc.org/edr/

1. https://docs.ogc.org/is/19-086r5/19-086r5.html
